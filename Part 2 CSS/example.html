This is paragraph. And this paragraph has a link that will send you to the Google home page.

This paragraph is located inside a div tag with a class called 'section.' The paragraph itself has a class of 'green.' In the stylesheet there is a rule reads '.section .green' that will apply specifically to this paragraph only.

This paragraph also has a class of green, but it does not have the same styling as the paragraph above. Why? In the stylesheet we declared '.section .green' instead of say '.green' or even '.section, .green.' Therefore there is no style for anything that just has a class of '.green.'

This paragraph has two classes ('section' and 'green'). Even though there is a rule in our stylesheet that reads '.section .green', this is not the style sheet that is applied. Instead, the one that is applied is the rule that read '.section.green' This is because '.section .green' is interpreted as any 'green' class that is inside an element with a 'section' class. While '.section.green' is interpreted as any element that has both a 'section' class AND a 'green' class.
